Διαφορές στη λειτουργία του Ηλ. Ταχυδρομείου

Η μέθοδος διαγραφής/μετακίνησης μηνυμάτων της εφαρμογής διαφέρει απο το WebMail. Όταν επιλέγετε ένα μήνυμα για διαγραφή ή μεταφορά το μήνυμα θα μεταφερθεί στο φάκελο που έχετε επιλέξει ή στα "Διαγραμμένα"(αν πρόκειτε για διαγραφή) και τυχόν μηνύματα που έχετε διαγράψει μέσα απο το Webmail αλλά δεν έχει γίνει οριστική διαγραφή("expunge") στο τρέχοντα φάκελο θα διαγραφούν οριστικά.

Όταν διαγράφετε ένα μήνυμα απο τα "Διαγραμμένα" αυτό διαγράφεται οριστικά μαζί με οποιοδήποτε άλλο μήνυμα έχετε διαγράψει μέσα απο το Webmail αλλά δεν έχει γίνει οριστική διαγραφή.

Τα μηνύματα που έχετε διαγράψει μέσα απο το Webmail αλλά δεν έχει γίνει οριστική διαγραφή εμφανίζονται στην εφαρμογή με γκρι χρώμα.

Passwords

Στο Τμήμα Πληροφορικής λειτουργούν μια πληθώρα από υπηρεσίες οι οποίες απαιτούν κάποιο κωδικό (password) για την πρόσβαση σε αυτές. Χαρακτηριστικό παράδειγμα είναι οι:

  • Πρόσβαση με SSH και FTP στον server aetos.it.teithe.gr
  • Dialup πρόσβαση
  • Λήψη email με POP3 και IMAP
  • Portal υπηρεσιών
  • Ηλεκτρονική Γραμματεία
  • Βαθμολογίες
  • Ηλεκτρονικές Δηλώσεις

Η πρόσβαση στις υπηρεσίες αυτές γίνεται με 3 χαρακτηριστικά για κάθε φοιτητή:
  • To username που έχει στον server aetos.it.teithe.gr
  • Τον αριθμό μητρώου που έχει στο Τμήμα
  • Το password που έχει παραλάβει

Το username και το password για τον server aetos.it.teithe.gr δίνονται από τον κ. Αμανατιάδη στην αίθουσα 209 (αίθουσα VAX).

Πόσα password υπάρχουν;

Ο κάθε φοιτητής είναι υποχρεωμένος να θυμάται ένα μόνο password για να έχει πρόσβαση στις παραπάνω υπηρεσίες.

Το Πληροφοριακό Σύστημα των Γραμματειών (pithia.teithe.gr) δεν είναι υπηρεσία του Τμήματος.

Πως γίνεται η πρόσβαση σε κάθε υπηρεσία;

Αν υποθέσουμε πως είστε ο φοιτητής με:
  • Αριθμό Μητρώου (ΑΜ) 03/1789 (ή για συντομία 031789)
  • Username: student1
  • Password: secret1
τότε έχετε πρόσβαση ως εξής:

SSH και SFTP στον server aetos.it.teithe.gr

Δίνετε σαν username το "student1" και σαν password το "secret1";

Dialup πρόσβαση

Στις ρυθμίσεις για dialup σύνδεση δίνετε σαν username το "student1@aetos" και σαν password to "secret1"

Λήψη email με POP3 και IMAP

Βάζετε σαν username το "student1" και σαν password το "secret1"

Λοιπές υπηρεσίες

Δίνετε σαν username το "031789" ή σκέτο το "1789" και σαν password το "secret1"

Αν υπάρχει πρόβλημα;

Σε οποιαδήποτε περίπτωση αν σαν εμφανίζεται ότι υπάρχει πρόβλημα με τον κωδικό σας (password), θα πρέπει να τον αλλάξετε χρησιμοποιώντας τις οδηγίες που βρίσκονται στις Χρήσιμες Πληροφορίες

Πληροφοριακό σύστημα γραμματείας

Οι λογαριασμοί και οι κωδικοί του πληροφορικού συστήματος της γραμματείας (pithia) διαχειρίζονται αποκλειστικά από τη γραμματεία. Για σχετικές πληροφορίες πρέπει να απευθύνεστε στη γραμματεία.
